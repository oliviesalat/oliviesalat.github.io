Research Assistant

A simple tool for exploring and extracting scientific knowledge

About the Project

Research Assistant is a lightweight research helper tool currently under development. It allows researchers and students to retrieve, extract, and visualize structured content from scientific publications such as formulas and metadata.

Technologies Used

  • Elsevier API for ScienceDirect
  • Python and Django (backend)
  • HTML/CSS and JS (frontend)

Status

The project is currently in a development phase and is intended for academic and personal research purposes.
